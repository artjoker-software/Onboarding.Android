# Usage 
Add it in your root build.gradle at the end of repositories:

    allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
	
Add the dependency:

	dependencies {
	        compile 'com.github.artjokerLLC:Onboarding.Android:f27e4f4a11'
	}
